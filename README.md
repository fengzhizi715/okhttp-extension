# okhttp-extension

## Features:

* 支持常见的请求方法：GET、POST、HEAD、PUT、PATCH、DELETE
* 支持 DSL 创建网络请求
* 支持自定义线程池
* 支持 Kotlin 协程
* 支持响应式(RxJava、Reactor) 
* 支持函数式
* 支持异步请求的取消
* 支持 Request、Response 的拦截器
