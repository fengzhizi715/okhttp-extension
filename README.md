# okhttp-extension

## Features:

* 支持 DSL 创建网络请求
* 支持自定义线程池
* 支持 Kotlin 协程
* 支持响应式(RxJava、Spring Reactor) 
* 支持函数式
* 支持异步请求的取消
* 支持 Request、Response 的拦截器
* 支持 traceId 的传递(通过拦截器)
* 支持 Retrofit 框架
